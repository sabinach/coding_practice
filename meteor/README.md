- MongoDB (back-end)
- Node.js (back-end)
- React (front-end)

## To Install Meteor
1. `curl https://install.meteor.com/ | sh`
2. `meteor create project-name`
   
## To Run 
1. `cd project-name`
2. Meteor:
  - `meteor`
  - Go to: `http://localhost:3000/`
3. Mongo:
  - `meteor mongo`
