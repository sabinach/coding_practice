## To Run MySQL:
Make sure MySQL Server is ON via the System Preferences Panel, then in terminal, run: 
```mysql -u root -p```      
```ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '1234';```     
```flush privileges;```
```CREATE DATABASE customers;```
```USE customers```
```SHOW TABLES;```
```SELECT * FROM customers;```
```DESCRIBE customers;```

## To Run Server:
```node server.js```
