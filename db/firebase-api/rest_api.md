# REST API

## Firebase DB Endpoints

| **URL** | **HTTP Verb**   |  **Action**|
|------------|-------------|------------|
| /api/items/                | GET            | read all items
| /api/items/:id            | GET             | read item (by ID)
| /api/items/:id            | POST          | create item
| /api/items/:id            | PUT             | edit item
| /api/items/:id            | DELETE      | delete item
