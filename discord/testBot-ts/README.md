# testBot-ts

### Specs
- NodeJS (>v16.6)
- Discord.js (v13)
- TypeScript

### Setup [[video](https://youtu.be/j_sD9udZnCk)]
- ```npm init```
- ```npm install discord.js <check package.json>```

### To Run
- ```npm ci```
- ```npm run start``` or ```npm run dev``` (for nodemon)

### Tutorial
- https://sabe.io/tutorials/how-to-build-discord-bot-typescript (create slash command)
