# testBot

### Specs
- NodeJS
- Discord.js

### Setup
- Instructions: [Code Your Own Discord Bot - Basics 2021](https://youtu.be/j_sD9udZnCk) by CodeLyon
  * ```npm init```
  * ```npm install discord.js``` (requires node version >16)

### To Run
- ```node index.js```
