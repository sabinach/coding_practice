# PyOpenGL

# To Run
```python filename.py```

# To Setup
- Download `.whl` files 
    * For Python 3.8 or 3.9 with amd64: https://github.com/TheDitis/SmartCue/tree/main
    * For Python 3.10 with amd64: https://github.com/RioT1133/raycasting/tree/master
- ```pip install PyOpenGL-***.whl```
- ```pip install PyOpenGL_accelerate-***.whl```
